Simon Game

Simon is a game in which the computer generates a random color ,
the user has to remember the pattern everytime and has to click the respectve buttons .
if the user fails , the game ends.

Instructions

  1. Start pressing any key on the keyboard.
  2. The machine will generate a random color (say blue), will be recognized with sound and blink effects.
  3. the user has to click on the blue color.
  4. the machine will again generates a color (say green).
  5. now the user has to click blue followed by green and repeat the process.
  6. the game ends when the user fails to click the correct sequence .
